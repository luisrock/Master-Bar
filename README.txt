Display a bar with any message or notification on top of your WordPress website. 

== Description ==
Master Bar plugin offers an easy way to display a bar with any message on top of your WordPress website. You can choose different messages for logged and non-logged users. You can select where the bar will appear: homepage, pages, posts, custom post types. You can even opt to use a shortcode [masterbar] in the page where you want the bar to show. Select  color, background-color, font-size, border and/or apply your custom css code. Use Google fonts. Choose to let users click to hide the bar. Translate it.

== Installation ==
1. Upload file to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Go to the Admin Dashboard and look for the MasterBar menu item.
4. Click ’On’ for ’Activate Bar’.
5. Follow the instructions to build the bar and place it where you want.
